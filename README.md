# social-media
📱 Social Media Web Application (Django)

-This is a simple social media web application built with Django for the backend and HTML/CSS for the frontend.
-The application allows users to create accounts, interact with posts, and connect with other users through a basic social networking system.
🚀 Key Features :
-🔐 User Authentication System (Register, Login, Logout)
-🔑 Password Reset & Forgot Password via Email
-✅ User Account Verification
-📧 Email Sending System
-👤 User Profile Creation and Profile Editing
-➕ Follow and Unfollow Other Users
-❤️ Like and Unlike Posts
-📝 Create and Publish Posts
-🔍 Search Posts by Title or Tags
🛠️ Technologies & Tools :
-Python
-Django
-HTML, CSS
-SQLite Database
